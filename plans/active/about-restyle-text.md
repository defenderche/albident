# План: Облегчение текстов на «О нас» (подход + гарантии)

## Цель
Сделать страницу «О нас» легче для чтения: убрать дублирующийся заголовок,
переверстать тяжёлый текст блока «О клинике» в скан-формат и сократить тексты
вокруг блока «Гарантии». Структура страницы (набор блоков) не меняется.

## Контекст
- `feature-spec-about.md` §1.1–1.3 — структура блоков сохраняется. Со спекой совпадает (правится контент + подача), спека не редактируется.

Решения с пользователем:
