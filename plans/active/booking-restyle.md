# План: Запись (/booking) в стиле «клиника-студия»

## Цель
Распространить язык дизайна на `/booking` — последняя группа внутренних страниц.

## Контекст
Язык установлен (главная, услуги, о нас, контакты, privacy). Skill
`albident-design-map`: прототип → согласование → код. Учесть аудит booking:
форма без контейнера (нужна карточка-зона), бедное success-состояние (нужны
галочка + «Заявка отправлена» + подпись). Поля и валидацию НЕ трогаем —
только визуал (форма уже работает: 7 полей, чекбокс согласия, Zod).

## Шаги
- [ ] 1. Дизайн-карта: прототип `/booking` — открытый hero (заголовок + интро),
      форма в карточке-зоне, выразительное success (иконка + заголовок + подпись).
- [ ] 2. Показать карту, согласовать (цикл правок).
- [ ] 3. После «да» — код: страница /booking (обёртка-секция) + BookingForm
      (карточка-контейнер + богатый success). Логику/валидацию не менять.
- [ ] 4. Проверка: tsc, lint, test, build; визуальная сверка с картой.

## Критерии готовности
- `/booking` в едином стиле; форма — весомая карточка; success выразительный.
- Карта и сайт совпадают. Зелёные lint / tsc / test.
