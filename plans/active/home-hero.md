# План: Доработка Hero главной

## Цель
Довести Hero главной до вида графической карты: двухколоночная карточка с
заголовком (акцентные слова), 3D-визуалом справа и полосой ключевых
показателей снизу.

## Контекст
Отложенные элементы из `design-system-port` (см. его «Открытые детали»).
Дизайн Hero уже в карте `design-map/index.html` (одобрен) — внедряем 1:1.
Визуал — 3D-челюсть (ChatGPT-рендер, прозрачный PNG 1536×1024) в
`public/hero-teeth.png`. См. `feature-spec-home §1.1`.

## Шаги
- [x] Картинка в `public/hero-teeth.png`
- [x] Hero: двухколоночная карточка (текст + визуал), `next/image` с
      `priority`/`sizes`, акцентные слова в заголовке (rich-перевод)
- [x] Полоса показателей под Hero — 4 шт., placeholder-значения из карты
      (12+ лет, 5 врачей, 8 направлений, 3 языка)
- [x] Переводы: `<accent>` вокруг «Albident» и `stats.*` в ru/en/tr
- [x] `build` / `lint` / `tsc` зелёные

## Критерии готовности
- Hero совпадает с картой: визуал справа, статистика снизу, синий акцент.
- Сборка зелёная, картинка оптимизируется через `next/image`.
