# План: Редизайн главной — «клиника-студия»

## Цель
Уйти от ощущения «вся информация в коробках на сером фоне» к открытой,
дышащей странице премиум-уровня: воздух, чередующиеся фоновые полосы, крупная
типографика, открытый hero, 3D-мотив, доверие на виду. Главная — флагман
нового сквозного языка дизайна.

## Контекст
Запрос пользователя + research сильных сайтов стоматологий (Awwwards: LAVA,
Klaas, Halo; обзоры delmain/unnus). Ограничение проекта: фото живых людей не
используются — опираемся на типографику, 3D-мотив, силуэты, воздух.
Работа по skill `albident-design-map`: сначала карта → согласование → код.

## Направление (утверждено пользователем)
Сквозные приёмы: разбоксить секции (открытые полноширинные полосы на
чередующемся фоне вместо карточек на сером), крупный вертикальный ритм,
выразительная типографика, 3D-зуб как мотив, тонкая анимация появления.

## Шаги
- [ ] 1. Дизайн-карта: новые сквозные токены при необходимости (фоновые полосы,
      ритм отступов, шкала заголовков).
- [ ] 2. Дизайн-карта: пересобрать организм главной — открытый hero (изображение
      выходит за сетку, без карточки-рамки) → услуги полосой → «почему мы»
      крупными цифрами → блок отзывов (новый на главной) → FAQ открытым списком
      → широкий финальный CTA.
- [ ] 3. Показать карту пользователю, согласовать (цикл правок).
- [ ] 4. После «да» — перенести в код: Hero, ServicesPreview, WhyUs, новый блок
      отзывов, Faq, FinalCta + токены globals.css.
- [ ] 5. Проверка: tsc, lint, test, build; визуальная сверка с картой.

## Критерии готовности
- Главная читается как цельная дышащая страница, а не стопка коробок.
- Карта и сайт совпадают.
- Зелёные lint / tsc / test.

## Открытые вопросы
- Блок отзывов на главной = правка спеки/карты (сейчас отзывы только на /about).
- Возможно вынести часть приёмов (токены) в отдельный последующий план для
  остальных страниц.
