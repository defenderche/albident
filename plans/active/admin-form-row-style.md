# План: Строки списков в админ-форме без вложенных коробок

## Цель
Убрать «коробку в коробке» у повторяемых строк (этапы / под-процедуры / FAQ):
вместо рамки вокруг каждого элемента — разделитель + подпись «Этап N» +
компактная ссылка «Удалить». Поля сохраняют одну рамку.

## Контекст
- Внутренний инструмент (`components/admin/ServiceForm.tsx`), не публичный сайт.
- Предпросмотр — живьём на ветке (форма уже работает).

## Шаги
- [ ] Переделать строки этапов / под-процедур / FAQ (без рамки-коробки, разделитель + «N» + «Удалить»)
- [ ] Согласовать вид с пользователем (живьём)
- [ ] Проверки: `tsc`/`lint`/`build`

## Критерии готовности
- Нет вложенных коробок; строки разделены линией, удаление — компактной ссылкой.
- Проверки зелёные.
