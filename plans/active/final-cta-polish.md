# План: Доработка финального CTA-блока

## Цель
Привести FinalCta к виду графической карты: добавить подзаголовок и
декоративный элемент, сделать блок компактнее (сейчас выглядит пустовато).

## Контекст
В `design-map/index.html` у CTA-баннера есть подзаголовок + декор (силуэт
зуба) — в код перенесли только заголовок и кнопку. Это возврат к карте
(одобрено), не новый дизайн. FinalCta — общий компонент, используется на
`/`, `/services`, `/about`, `/contacts` (namespace `*.finalCta`).

      чуть компактнее по высоте

- [ ] `build` / `lint` / `tsc` зелёные

## Критерии готовности
- CTA-блок совпадает с картой: заголовок + подзаголовок + кнопка + декор.
- Сборка зелёная, тексты на 3 языках.
