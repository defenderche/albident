# План: Гарантии в коде (процесс на /about + чат)

## Цель
Перенести спек-правки по гарантийному процессу в код: на /about вводный
параграф описывает процесс для уехавшего пациента и появляется блок общих
исключений; чат может цитировать процесс гарантии, но не выносит вердикт по
конкретному случаю.

## Контекст
Спеки: `feature-spec-about.md §1.3`, `global-spec.md §10.1`,
`feature-spec-chat.md §6`. Модель: дистанционная оценка по WhatsApp/email +
приезд в Стамбул для ручной работы (работа бесплатна, дорога нет). Общие
исключения из §10.1: травмы, несоблюдение рекомендаций/гигиены, пропуск
контрольных визитов, естественный износ, вмешательство сторонних врачей.
Текущее: intro/footnote гарантий — i18n (`About.warranties`), данные по услугам
— `content/warranties.ts`; чат-промпт — `lib/chat/prompt.ts`, контекст —
`lib/chat/context.ts`.
