# План: Ссылка на врача со страницы услуги → блок «Команда»

## Цель
Ссылка «Услугу ведёт: …» на странице услуги ведёт к блоку «Команда» на
`/about` (а не на верх страницы, откуда приходится листать вниз).

## Контекст
- `feature-spec-service-page.md` §1.1 — врач упоминается одной строкой, без карточки и без секции «Связанные врачи». Сохраняем как есть, меняем только цель ссылки.
- `feature-spec-about.md` §1.4 — блок «Команда».
- Спека молчит про поведение якоря (вердикт «в») — локальное решение.
- Не изменение вида (нет новых UI-элементов) → design-map не нужен.

## Шаги
- [x] `AboutTeam.tsx` — `id="team"` + `scroll-mt-24` на секцию (чтобы заголовок не уезжал под sticky-header).

- Клик по строке врача на `/ru/services/[slug]` скроллит к блоку «Команда».
- Заголовок «Команда» виден с отступом под шапкой.
- `npx tsc --noEmit` чист.
