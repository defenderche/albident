# План: Графическая карта (дизайн-язык)

## Цель
Зафиксировать визуальный язык будущего редизайна Albident в одном
самодостаточном HTML-артефакте: дизайн-токены + каталог компонентов в стиле
референса. Карта станет опорой при переделке реальных страниц сайта.

## Контекст
Источник — внешний референс дентал-лендинга (стиль: синий акцент, светлые
карточки, крупные скругления, мягкие тени, 3D-визуал, много воздуха).
Адаптируем **визуальный язык**, не структуру: раскладка страниц берётся из
наших спек (`functional-map.md`, `feature-spec-home.md`), не из референса.

Согласованные решения по адаптации:
- Карта = дизайн-язык + каталог компонентов; структура страниц — из спек.
- Блок статистики добавляем (сдержанный доверие-сигнал).
- Финальный CTA — синий акцент (фон CTA в спеке больше не зафиксирован —
  убран из `functional-map.md`, commit 5cac924).
- Фото реальных людей не используются: врачи — силуэты с инициалами,
  авторы отзывов — монограммы, декор CTA — 3D-элемент (не персона).
- Навигация: Главная / Услуги / О нас / Контакты + переключатель языка +
  CTA «Записаться». Соцсети — Instagram / WhatsApp / Telegram. App-бейджи,
  блог, Facebook — убраны (вне MVP / вне спеки).

## Расширение (по запросу): полная дизайн-система
Карта расширена с «дизайн-языка» до полной UI-системы, покрывающей весь
функционал сайта. Подход: дизайн придуман по стилю фото-референса и
масштабирован на все 7 страниц + чат + формы (включая то, чего в референсе
не было). Чистый UI, без UX-поведения.
- [x] **Foundations** — палитра (акцент/нейтрали/семантика форм), типографика
      (display→eyebrow), скругления, тени, отступы, иконостиль
- [x] **Atoms** — кнопки (варианты/размеры/состояния), бейджи/чипы/ценник,
      поля формы (input/textarea/select/checkbox/radio + states), аватар/силуэт/
      иконка-плитка, логотип, язык, соцсети, крошки, элемент статистики
- [x] **Molecules** — карточки (услуга/врач/отзыв/преимущество), FAQ-аккордеон,
      строки гарантий, статистика, навигация, мобильное меню, контакт-строка,
      чат-пузыри/ввод, алерты/тост
- [x] **Organisms** — header, hero+статистика, шапка страницы, гриды услуг,
      преимущества, FAQ-секция, команда, отзывы, форма записи (+success),
      финальный CTA, чат-виджет, footer

## Критерии готовности
- HTML открывается в браузере и показывает всю систему (токены → компоненты →
  блоки) без сборки.
- Каждый адаптированный относительно референса элемент помечен.
- Спека главной приведена в соответствие (блок статистики).
