# План: Внешний вид админ-формы (плоский, без вложенных коробок)

## Цель
Убрать «коробку в коробке» в форме услуги: плоские секции без карточек,
повторяемые строки без рамок, читаемые цвета.

## Контекст
- Внутренний инструмент (`components/admin/ServiceForm.tsx`, `app/admin/layout.tsx`),
  не публичный сайт. Предпросмотр — живьём на ветке.

## Шаги
- [x] Строки этапов / под-процедур / FAQ без рамки-коробки: разделитель (`divide-y`) +
      подпись «Этап N» + компактная ссылка «Удалить»
- [x] Плоские секции: убрать карточку (рамка/тень/фон), разделять заголовком + линией;
      добавлен заголовок «Основное»
- [x] Смена цветов: фон админки белый, поля ввода — серые (цвет прежнего фона),
      через тонкие обёртки `FieldInput`/`FieldTextarea` (scoped, не трогает публичные Input/Textarea)
- [x] Согласовано с пользователем живьём
- [x] Проверки: `tsc`/`lint`/`build` — зелёные

## Критерии готовности
- Один уровень рамок (только поля); секции плоские; фон/поля поменяны местами.
- Проверки зелёные.
