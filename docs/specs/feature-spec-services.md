# Feature Spec — Обзор услуг (`/services`)

Обзорная страница со списком всех услуг клиники в виде карточек. Точка входа в раздел услуг — отсюда пациент переходит на детальные страницы конкретных процедур.

---

## 1. Структура страницы

Страница состоит из 3 блоков, расположенных сверху вниз:

### 1.1 Hero

- Заголовок страницы (например, «Наши услуги»).
- Краткое описание (1–2 строки) — общая характеристика спектра услуг клиники.

### 1.2 Сетка карточек услуг

Все 8 услуг представлены карточками в виде сетки. Услуги:

- Терапия
- Хирургия
- Имплантация
- Протезирование
- Ортодонтия
- Эстетика
- Гигиена / профилактика
- Пародонтология

**Состав карточки услуги:**

- Название услуги
- Краткое описание (1–2 строки)
- Цена «от»
- Ссылка/кнопка «Подробнее» — ведёт на детальную страницу услуги `/services/[slug]`

### 1.3 CTA-блок

Кнопка «Записаться» в конце страницы — для пациента, который уже определился с записью, не уходя на детальную страницу.

## 2. Точки выхода

- **Карточка услуги** → `/services/[slug]` — детальная страница конкретной услуги.
- **CTA «Записаться»** → `/booking` — форма записи **без** предзаполненной услуги. Пациент сам выбирает услугу в форме (или оставляет пустым).

## 3. SEO

Страница имеет уникальные мета-теги:

- **`title`** — индивидуальный для страницы `/services`.
- **`description`** — индивидуальное краткое описание (~150 символов).

Конкретные тексты определяются на этапе подготовки контента.
