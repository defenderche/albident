# Feature Spec — Контакты (`/contacts`)

Страница с контактной информацией клиники: адрес, способы связи, часы работы. Используется пациентами, которым нужно связаться с клиникой напрямую или быстро найти данные о расположении.

Глобальные элементы (header и footer) описаны в `functional-map.md` и здесь не дублируются.

---

## 1. Структура страницы

Страница состоит из 4 блоков, расположенных сверху вниз:

### 1.1 Hero

- Заголовок страницы (например, «Контакты»).
- Короткое intro (1 строка) — приглашение связаться удобным способом.

### 1.2 Контактные данные

Блок с основными контактами клиники:

- **Адрес** (район Şişli, Стамбул; конкретная улица — на этапе контента).
- **Телефон / WhatsApp** — кликабельный (`tel:` для вызова, `wa.me/` для открытия WhatsApp).
- **Email** — кликабельный (`mailto:` для открытия почтового клиента).

### 1.3 Часы работы

Расписание работы клиники по дням недели.

### 1.4 CTA-блок

Финальный призыв к записи. Кнопка «Записаться» → `/booking`.

## 2. Точки выхода

- **CTA-блок** → `/booking`.
- **Кликабельные контакты** в блоке «Контактные данные» — открывают приложение для звонка / WhatsApp / почтовый клиент. Это не навигация по сайту, но стандартное поведение клика на контакт.

## 3. SEO

Страница имеет уникальные мета-теги:

- **`title`** — индивидуальный (например, «Контакты — стоматологический центр Albident»).
- **`description`** — индивидуальное краткое описание (~150 символов).

Конкретные тексты определяются на этапе подготовки контента.
