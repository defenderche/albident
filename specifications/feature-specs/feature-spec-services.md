# Feature Spec — Обзор услуг (`/services`)

Обзорная страница со списком всех услуг клиники в виде карточек. Точка входа в раздел услуг — отсюда пациент переходит на детальные страницы конкретных процедур.

---

## 1. Структура страницы

Страница состоит из 2 блоков, расположенных сверху вниз:

### 1.1 Hero

- Заголовок страницы (например, «Наши услуги»).
- Краткое описание (1–2 строки) — общая характеристика спектра услуг клиники.

### 1.2 Сетка карточек услуг

Все 8 услуг представлены карточками в виде сетки. Услуги:

- Терапия
- Хирургия
- Имплантация
- Протезирование
- Ортодонтия
- Эстетика
- Гигиена / профилактика
- Пародонтология

**Состав карточки услуги:**

- Название услуги
- Краткое описание (1–2 строки)
- Диапазон цен «от $X — до $Y» (USD, см. `../global-spec.md §9`)
- Ссылка/кнопка «Подробнее» — ведёт на детальную страницу услуги `/services/[slug]`

## 2. Точки выхода

- **Карточка услуги** → `/services/[slug]` — детальная страница конкретной услуги.
- **Sticky header «Записаться»** → `/booking` — доступен с любого места страницы. Финального CTA-блока в конце страницы нет: для пациента, готового записаться, есть кнопка в шапке; для остальных карточки ведут на детальную страницу с прицельной информацией.

## 3. SEO

Страница имеет уникальные мета-теги:

- **`title`** (RU): «Услуги стоматологии — Albident, Стамбул».
- **`description`** (RU): «Все услуги стоматологии Albident: имплантация, виниры, ортодонтия, терапия, гигиена и др. Цены в USD, прозрачно. Запись на приём онлайн.»

EN и TR — переводы при наполнении контентом.
