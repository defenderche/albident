# Feature Spec — О нас (`/about`)

Страница раскрывает клинику пациенту: позиционирование, гарантии, команду врачей и отзывы. Используется для построения доверия и убеждения пациента — особенно международного, который выбирает клинику дистанционно.

Глобальные элементы (header и footer) описаны в `../functional-map.md` и здесь не дублируются.

---

## 1. Структура страницы

Страница состоит из 6 блоков, расположенных сверху вниз:

### 1.1 Hero

- Заголовок страницы (например, «О клинике Albident»).
- Короткое intro — суть клиники в 1–2 строках.

### 1.2 О клинике

Текстовый блок про клинику: подход, оборудование, философия работы. Раскрывает позиционирование «качество и доверие» из `../global-spec.md`.

### 1.3 Гарантии

Блок с упоминанием гарантий на основные виды работ. Конкретные сроки определяются на этапе подготовки контента.

### 1.4 Команда

Раздел с карточками 5 врачей клиники.

**Состав карточки врача:**

- Силуэт с инициалами (вместо фотографии — см. ограничение в `../global-spec.md`)
- ФИО
- Роль / специализация
- Достижения (опыт, обучение, сертификации)

Карточки используются также в блоке «Связанные врачи» на страницах услуг (см. `feature-spec-service-page.md`).

### 1.5 Отзывы

Раздел с отзывами пациентов. Статичные текстовые отзывы, читаются прямо на странице.

### 1.6 CTA-блок

Финальный призыв к записи в конце страницы. Кнопка «Записаться» → `/booking`.

## 2. Точки выхода

Единственная точка выхода со страницы — **CTA-блок** → `/booking`.

Остальные элементы (карточки врачей, отзывы, текстовые блоки) — informational, без интерактива.

## 3. SEO

Страница имеет уникальные мета-теги:

- **`title`** — индивидуальный (например, «О клинике Albident»).
- **`description`** — индивидуальное краткое описание (~150 символов).

Конкретные тексты определяются на этапе подготовки контента.
