# Feature Spec — О нас (`/about`)

Страница раскрывает клинику пациенту: позиционирование, гарантии, команду врачей и отзывы. Используется для построения доверия и убеждения пациента — особенно международного, который выбирает клинику дистанционно.

Глобальные элементы (header и footer) описаны в `../functional-map.md` и здесь не дублируются.

---

## 1. Структура страницы

Страница состоит из 6 блоков, расположенных сверху вниз:

### 1.1 Hero

- Заголовок страницы (например, «О клинике Albident»).
- Короткое intro — суть клиники в 1–2 строках.

### 1.2 О клинике

Текстовый блок про клинику: подход, оборудование, философия работы. Раскрывает позиционирование «качество и доверие» из `../global-spec.md`.

### 1.3 Гарантии

Блок раскрывает гарантийную политику клиники в трёх частях:

1. **Вводный параграф** — что значит «гарантия» в этой клинике и как выглядит гарантийный случай для уехавшего пациента: куда писать (WhatsApp / email), кто и как решает (дистанционная оценка врачом по фото / снимкам), нужно ли приезжать (ручная работа — приезд в Стамбул, сама работа бесплатна, дорога — нет). Факты — из `../global-spec.md §10.1`.
2. **Список гарантий** — для каждой категории работ: название → срок → короткое пояснение (что именно покрывает гарантия, что нет). Сроки — из `../global-spec.md §10`. Подача — definition-list без коробки/таблицы, чтобы вписаться в magazine-flow страницы. Под списком — короткий перечень **общих исключений** (что не покрывается ни по одной услуге), из `../global-spec.md §10.1`.
3. **Условия сохранения гарантии** — короткий футер: что должен делать пациент, чтобы гарантия осталась в силе (контрольные визиты, базовая гигиена).

### 1.4 Команда

Раздел с карточками 5 врачей клиники.

**Состав карточки врача:**

- Рисованный аватар — стилизованная AI-иллюстрация, не фото реального человека (см. ограничение в `../global-spec.md §12`). Если аватар отсутствует — fallback на силуэт.
- ФИО
- Роль / специализация
- Достижения (опыт, обучение, сертификации)

Карточки используются также в блоке «Связанные врачи» на страницах услуг (см. `feature-spec-service-page.md`).

### 1.5 Отзывы

Раздел с отзывами пациентов. Статичные текстовые отзывы, читаются прямо на странице.

### 1.6 CTA-блок

Финальный призыв к записи в конце страницы. Кнопка «Записаться» → `/booking`.

## 2. Точки выхода

Единственная точка выхода со страницы — **CTA-блок** → `/booking`.

Остальные элементы (карточки врачей, отзывы, текстовые блоки) — informational, без интерактива.

## 3. SEO

Страница имеет уникальные мета-теги:

- **`title`** (RU): «О клинике Albident — стоматология в Şişli, Стамбул».
- **`description`** (RU): «Стоматологический центр Albident в Şişli (Стамбул): команда врачей с международной квалификацией, гарантии, отзывы пациентов.»

EN и TR — переводы при наполнении контентом.
